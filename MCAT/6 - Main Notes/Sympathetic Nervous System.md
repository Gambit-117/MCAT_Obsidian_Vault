# Sympathetic Nervous System

The sympathetic nervous system is responsible for "fight or flight." The most common physiological changes include pupils dilating, decreasing in salivation, increase in respiratory and heart rate, increased glucose release from the liver, stimulated adrenal glands, and decreased digestion.
